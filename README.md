## 💡 About

- [TOS](https://gist.github.com/fiezt1492/d0c64a6bea28821a737f5c543d7cff64)

Sudo MeLy is a private Discord bot made only for Code MeLy Community Discord Server.

## ✨ Features

- Greeting System
- Leveling System
- Currency System
- Little Moderation Module
